SASS is A CSS Preprocessor

It prepares your styles for output as CSS. 

.fancy {
    border: 1px solid blue;
    &.highlight {
            background: yellow;
    }
}

.fancy {
    border: 1px solid blue;
}
.fancy.highlight {
    background: yellow;
}

Why SASS?

  1. Not entirely unlike CSS
  2. More organized code
  3. Variables ($maroon: #7a0019)
  4. Control structures (if...then)
  5. Loops (for $i from 1 through 42)
  6. Mixins! (Compass, Bourbon, Neat)

Need to Upgrade? Don't Panic!

gem update sass

How to know if an update is available?

# Check the version of all installed gems
gem list

# List gems with a more recent version available
gem outdated

# Install a specific version of a gem
gem install sass -v 3.3.1

Convert Existing CSS

The Harmless Way

Valid CSS is valid SCSS. 

cp styles.css styles.scss

Converting Existing CSS

The Mostly Harmless Way

sass-convert styles.css styles.scss

Now Minify!

sass --update styles.scss:styles.min.css --style compressed

You're Already Winning!

  • style.css: 43,362 bytes
  • style.min.css: 27,823 bytes
  • 36% smaller!
  • Your first step toward a Minimum Viable Mobile User Experience

SASS Nesting

Styles nested within styles will apply to child elements. 

.parent {
    /* Parent styles */
    color: red;

    .child {
        /* Child styles */
        color: blue;
    }
}
/* Generated styles */
.parent {
  /* Parent styles */
  color: red;
}
.parent .child {
  /* Child styles */
  color: blue;
}

Media Queries

Media queries can be nested

.container {
    /* Let's do this mobile first */
    width: 100%;

    @media screen and (min-width: 600px) {
        /* Give it a little room to breathe */
        width: 500px;
        margin: 0 auto;
    }
}
/* Generated styles */
.container {
    width: 100%;
}
@media screen and (min-width: 600px) {
    .container {
        width: 500px;
        margin: 0 auto;
    }
}

Parent Selectors

.parent {
    /* Parent styles */
    color: red;

    /* Use & to reference the parent selector */
    &:hover {
        background: green;
    }
}
.parent {
  /* Parent styles */
  color: red;
}
.parent:hover {
  background: green;
}

Parent Selector as Prefix (new in 3.3)

.button {
    &-normal {
        color: green;
    }
    &-error {
        color: red;
    }
}
.button-normal {
    color: green;
}
.button-error {
    color: red;
}

Variables ($)

/* Define colors in one place */
$beige: blanchedalmond;
$dark-blue: #483D8B;

body {
    background: $beige;
    color: $dark-blue;
}
body {
  background: blanchedalmond;
  color: #483D8B;
}

Variables: fonts

/* Define your font stack */
$fonts: Arial, Helvetica, sans-serif;
$monospace: "Source Code Pro", monospace;

body {
    font: 100% $fonts;
}
pre {
    font-family: $monospace;
}
body {
  font: 100% Arial, Helvetica, sans-serif;
}

pre {
  font-family: "Source Code Pro", monospace;
}

Variables: math

$overall-width: 960px;
$overall-height: 600px;
$main-content-width: $overall-height / $overall-width * 100%;
$meta-content-width: ($overall-height / 2) / $overall-width * 100%;

.main {
    width: $main-content-width;
}
.meta {
    width: $meta-content-width;
}
.main {
  width: 62.5%;
}

.meta {
  width: 31.25%;
}

Control Directives

$section-colors: #2e500b, #0a4b4b, #640e3f;

@for $i from 1 through 3 {
    #id#{$i} {
        background-color: nth($section-colors, $i);
    }
}
                        
#id1 {
    background-color: #2e500b;
}
#id2 {
    background-color: #0a4b4b;
}
#id3 {
    background-color: #640e3f;
}

Bourbon

Never look up another vendor prefix.

Installing Bourbon

gem install bourbon
bourbon install --path my/custom/path

Using Bourbon

/* Import the mixins */
@import 'bourbon/bourbon';

.gradient-example {
    /* Include the mixin in your styles */
    @include linear-gradient(#1e5799, #2989d8);
}
/* Get delicious CSS */
.gradient-example {
    background-color: #1e5799;
    background-image: -webkit-gradient(linear, left top, left bottom, color-stop(0%, #1e5799), color-stop(100%, #2989d8));
    background-image: -webkit-linear-gradient(top, #1e5799, #2989d8);
    background-image: -moz-linear-gradient(top, #1e5799, #2989d8);
    background-image: -ms-linear-gradient(top, #1e5799, #2989d8);
    background-image: -o-linear-gradient(top, #1e5799, #2989d8);
    background-image: linear-gradient(top, #1e5799, #2989d8);
}

Neat

A semantic grid framework.

Installing Neat

Install it on your computer: 

gem install neat

Install it in your project: 

bourbon install #If not installed
neat install

Using Neat: breakpoints

/* Import neat */
@import "../css/bourbon/bourbon"; /* Bourbon first. */
@import "../css/neat/neat";

/* Define breakpoints */
$mobile: new-breakpoint(max-width 960px 1);

.main {
    @include media($mobile) {
        float: none;
        width: 100%;
    }
}
@media screen and (max-width: 960px) {
  .main {
    float: none;
    width: 100%;
  }
}

Using Neat: grids

$column: 90px;
$gutter: 30px;
$grid-columns: 10;
$max-width: em(1088);

section {
    @include outer-container;
}

article {
    @include span-columns(6);
}

Compass

An open-source CSS Authoring Framework

Installing Compass

gem install compass
# Create new project
compass create path/to/project

# Already existing project
cd path/to/project
compass init
# Use compass commands instead of native sass
compass compile

# Watch for changes
compass watch

Using Compass

/* Import the mixins */
@import 'compass';

img:hover {
/* Include the mixin in your styles */
    @include transform(scale(120%, 120%) rotate(30deg));
}
/* Get vendor-prefixed CSS3 */
img:hover {
    -webkit-transform: scale(150%, 120%) rotate(30deg);
    -moz-transform: scale(150%, 120%) rotate(30deg);
    -ms-transform: scale(150%, 120%) rotate(30deg);
    -o-transform: scale(150%, 120%) rotate(30deg);
    transform: scale(150%, 120%) rotate(30deg);
}

Helper functions

  • Color math (adjust-lightness, adjust-saturation, tint)
  • Base-64 image encoding (inline-image)
  • Read image data (image-width, image-height)
  • elements-of-type
  • Sprites
# Compass is picky about paths
images/icons/facebook.png               images/icons/twitter.png
images/icons/facebook_hover.png         images/icons/twitter_hover.png
# Generate stylesheet
compass sprite "images/icons/*.png"
    # creates sass/sprites/_icons.scss
/* import into your styles */
@import "sprites/icons";
@include all-icons-sprites();
/* Generated styles */
.icons-sprite, .icons-facebook, .icons-twitter {
  background: url('/images/icons-s35d7438fab.png') no-repeat;
}
.icons-facebook {
  background-position: 0 -192px;
}
<-- Use the generated classes in your markup -->
<div class="icons-facebook"></div>
<div class="icons-twitter"></div>
