SASS

Syntactically Awesome Style Sheets

SASS is A CSS Preprocessor

It prepares your styles for output as CSS.

This: 

.fancy {
    border: 1px solid blue;
    &.highlight {
            background: yellow;
    }
}

Generates this: 


.fancy {
    border: 1px solid blue;
}
.fancy.highlight {
    background: yellow;
}

Why SASS?

  1. More organized code
  2. Variables ($maroon: #7a0019)
  3. Control structures (if...then)
  4. Loops (for $i from 1 through 4)
  5. Mixins! (Compass, Bourbon, Neat)

Convert Existing CSS

The Easy Way

Valid CSS is valid SCSS. 

cp styles.css styles.scss

Converting Existing CSS

The Slightly Harder Way

sass-convert styles.css styles.scss

Now Minify!

sass --update styles.scss:styles.min.css --style compressed

You're Already Winning!

  • style.css: 43,362 bytes
  • style.min.css: 27,823 bytes
  • 64% smaller!
  • Your first step toward a Minimum Viable Mobile User Experience

More Output Styles

/* nested (default) */
#main {
  color: #fff;
  background-color: #000; }
  #main p {
    width: 10em; }

More Output Styles

/* expanded */
#main {
  color: #fff;
  background-color: #000;
}
#main p {
  width: 10em;
}

More Output Styles

/* compact */

#main { color: #fff; background-color: #000; }
#main p { width: 10em; }

More Output Styles

/* compressed */

#main{color:#fff;background-color:#000}#main p{width:10em}

Watch While You Work

sass --watch styles.scss:styles.min.css --style compressed

Watch An Entire Directory

sass --watch path/to/sass/files/ --style compressed

Variables

/* Define colors in one place */
$heading-color: rgb(122, 0, 25);

/* Define values for math fun! */
$default-margin-value: 20;

/* Concatenate values */
$default-margin: 20 + px;

/* Do math! */
$width: 960 - ($default-margin * 2);

Import Other SASS Files

This: 

@import "path/to/settings";

Imports this file: 

path/to/_settings.scss

SASS Nesting

Styles nested within styles will apply to child elements. 

/* Parent element */
.parent {
    color: red;

    /* Child element */
    .child {
        color: blue;
    }
}
/* Generated styles */
.parent {
    color: red;
}
.parent .child {
    color: blue;
}

Parent Selectors

/* Reference the parent selector */
.parent {
    color: red;
    &:hover {
        background: #ccc;
    }
}
/* Generated styles */
.parent {
    color: red;
}
.parent:hover {
    background: #ccc;
}
